# 163 Music for Watch (云音乐手表版)

适用于小天才手表的网易云音乐播放器。

## 功能特性

- 🎵 **音乐搜索** - 搜索网易云音乐曲库
- ▶️ **音乐播放** - 支持播放控制（上一首/下一首/暂停/播放）
- 🔊 **音量调节** - 便捷的音量增减按钮
- ❤️ **收藏功能** - 本地收藏喜欢的歌曲
- 🔑 **Cookie登录** - 设置Cookie实现VIP音乐播放
- ☁️ **云端同步** - 登录后可使用云端收藏列表

## 技术要求

- Android 7.0+ (API 24)
- 目标: Android 8.0 (API 26)
- 屏幕: 360x320 (小天才手表)

## 构建

```bash
./gradlew assembleDebug
```

APK输出路径: `app/build/outputs/apk/debug/app-debug.apk`

## 使用说明

1. 安装APK到手表
2. 打开应用，点击右上角 **⋮** 进入更多界面
3. 在更多界面中搜索歌曲，点击即可播放
4. 如需播放VIP音乐，点击 **⚙** 进入设置，输入网易云音乐Cookie
5. 点击 **♡** 收藏喜欢的歌曲
